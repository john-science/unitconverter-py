# Build platform binary
* run in terminal : `pip install pyinstaller`
* run in terminal : `pyinstaller --onefile app.py`
* in folder dist, run `app.exe`

# direct run
* run in terminal : `python app.py`

# Testing
* run in terminal : `pip install pytest`
* next line  `pytest`
